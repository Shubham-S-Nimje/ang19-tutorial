Components

In Angular, a component is a building block of the UI. Every part of an Angular app like headers, menus, forms, or footers is made using components.

A component has three main parts:

  • HTML file - decides how it looks (template)
  • CSS file - decides how it is styled
  • TypeScript file - decides how it works (logic)

How to Create a Custom Component

You can create your own component using Angular CLI. Open your terminal and run: 

ng generate component component-name

Example:

ng generate component header

This will create four files:

  • header.component.ts
  • header.component.html
  • header.component.css
  • header.component.spec.ts (for testing)

How to Use the Component

You can use your custom component in any other template using its selector. For example: 

<app-header></app-header>

That’s it! You’ve created and used a custom component in Angular.
