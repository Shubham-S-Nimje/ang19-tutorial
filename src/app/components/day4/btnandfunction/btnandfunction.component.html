Buttons and Functions

{{ number }}

{{ text }}

In this example, we use a button to increase a number and call a function when it reaches 5.

What Happens:

  • The value of number starts from 0.
  • Every time you click the button, number increases by 1.
  • When the number reaches 5, another function is called that:
    • Resets the number back to 0
    • Shows the message: "Other function called"
  • If the number is not 5, the message is empty.

HTML Code:


<section>
  <h2>Buttons and Functions</h2>
  <p>{{ number }}</p>
  <button (click)="onClick()">Click</button>
  <p>{{ text }}</p>
</section>

This is a simple way to use functions and events in Angular.
